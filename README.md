## Katkıda Bulunma

Eğer katkıda bulunmak isterseniz, aşağıdaki adımları takip edebilirsiniz:

1. Repo'yu fork edin ve kendi GitHub hesabınıza kopyalayın.
2. İlgili değişiklikleri yapmak için kopyaladığınız repo'yu düzenleyin.
3. Yaptığınız değişiklikleri açıklayan açıklayıcı bir açıklama ile commit yapın.
4. Yaptığınız değişiklikleri kopyaladığınız repo'ya push edin.
5. Orijinal repoya pull request (PR) göndererek değişikliklerinizi önerin.
